# جدول اولویت‌بندی طراحی صفحات و componentها

| Priority | صفحات اول طراحی | Componentهای اول طراحی | دلیل |
| --- | --- | --- | --- |
| **P0 — Core experience** | Home، Search، Search Results، Podcast Detail، Episode Detail، Mini Player، Full Player، Library basic، Login/Register | Player controls، Episode card، Podcast card، Search input، Content rail، Save/Follow، Skeleton/Error/Empty | بدون این‌ها محصول listener ارزش اصلی ندارد. |
| **P0 — Creator core** | Creator onboarding، Creator dashboard، Podcast settings basic، Episode upload، Episode editor، Publish/schedule، Processing status | Upload dropzone، Episode form، Publish checklist، Processing timeline، Status badge، Toast | بدون publish flow محتوایی تولید نمی‌شود. |

| **P1 — Power listening** | Queue، Playlists، Downloads، History، Transcript، Chapters | Queue panel، Playlist editor، Download button، Transcript viewer، Chapter list، Speed selector، Sleep timer | retention و differentiation برای power users. |
| **P1 — Creator growth** | Analytics overview، Episode analytics، Audience analytics، Revenue analytics، Payouts | Metric card، Chart card، Insight card، Payout table، Export controls | creator retention و اعتماد به درآمد. |
| **P2 — Enterprise depth** | Campaigns/Ads، Team members advanced، Copyright cases، Refund review، Entitlement override، Feature flags، System health | Risk score، Bulk actions، Advanced filters، Approval workflow، Feature flag table، Incident banner | برای enterprise scale لازم، اما MVP را پیچیده می‌کند. |
| **P2 — Differentiators** | Clips/snippets، public creator profile، advanced recommendations، community/engagement، dynamic artwork themes | Clip selector، Waveform editor، Recommendation reason label، Comment module، Dynamic color accent | ارزش افزوده، اما بعد از core reliability. |

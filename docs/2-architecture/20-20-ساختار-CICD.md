## pipeline پیشنهادی

```
```
Pull Request:  
1. lint  
2. unit tests  
3. proto breaking check  
4. event schema validation  
5. affected services build  
6. docker build dry-run  
7. migration check  
  
Merge to main:  
1. build images  
2. push images  
3. update manifests  
4. ArgoCD sync dev  
  
Tag release:  
1. deploy staging  
2. smoke test  
3. canary prod  
4. progressive rollout

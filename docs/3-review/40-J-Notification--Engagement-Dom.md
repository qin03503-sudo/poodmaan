# J. Notification & Engagement Domain

## 3.45 Notification Service

### وظیفه

ارسال اطلاع‌رسانی.

### کانال‌ها

-   push
    
-   email
    
-   in-app
    
-   SMS در صورت نیاز
    

### مسئولیت‌ها

-   event-based triggers
    
-   delivery orchestration
    
-   retry
    
-   template rendering
    
-   localization
    

### ارتباطات

-   Kafka events
    
-   preference service
    
-   SES / FCM / APNs
    
-   creator release events
    
-   billing reminders

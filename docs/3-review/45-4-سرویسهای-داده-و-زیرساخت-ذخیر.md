# 4) سرویس‌های داده و زیرساخت ذخیره‌سازی

## 4.1 PostgreSQL Clusters

برای داده‌های relational و transactional:

-   users
    
-   catalog
    
-   billing
    
-   library
    
-   playlists
    
-   moderation state
    

### نیازها

-   primary + read replicas
    
-   backup
    
-   PITR
    
-   partitioning for large tables
    
-   migration framework

## 4.3 Kafka Cluster

هسته ارتباطات async.

### کارکردها

-   decoupling
    
-   event sourcing-like patterns در برخی دامنه‌ها
    
-   analytics ingestion
    
-   CDC delivery
    
-   background workflow triggers
    

### نیازها

-   schema registry
    
-   DLQ
    
-   lag monitoring
    
-   multi-AZ deployment
